# CSVision (React + TypeScript + Vite)

## Run locally

1. Install dependencies:
   npm install
2. Start dev server:
   npm run dev
3. Open the URL shown in terminal.

## What this app includes

- CSV file upload
- Table rendering from CSV headers/rows
- Per-column text filters
- Click-to-sort per column (asc, desc, off)
- Row count after filtering
